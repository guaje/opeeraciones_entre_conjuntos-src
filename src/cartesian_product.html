## Producto cartesiano

Producto cartesiano: Definición

El conjunto producto cartesiano de dos conjuntos \(A\) y \(B\), es el conjunto de todas las parejas ordenadas, cuya primer componente es un elemento del conjunto \(A\) y cuya segunda componente es un elemento del conjunto \(B\). El conjunto producto cartesiano es único y se nota \(A \times B\). El conjunto producto cartesiano puede ser definido mediante el siguiente predicado:
$$\phi_{A \times B} ((x, y)) \Leftrightarrow (x, y) \in A \times B := (x \in A) \land (y \in B)$$

Utilizando notación por comprensión
$$A \times B = \{(x, y) : (x \in A) \land (y \in B)\}$$

Producto cartesiano: Ejemplo

Para los conjuntos \(A = \{"Cara", "Sello"\}\) y \(B = \{"COP", "USD"\}\) se tiene que:

$$A \times B = \{("Cara", "COP")), ("Cara", "USD"), $$ $$("Sello", "COP"), ("Sello", "USD")\}$$
$$B \times A = \{("COP", "Cara")), ("COP", "Sello"), $$ $$("USD", "Cara"), ("USD", "Sello")\}$$

Producto cartesiano: Ejemplo

Para los conjuntos \(A = \{J, Q, K\}\) y \(B = \{\color{blue}{\heartsuit}, \color{black}{\spadesuit}\}\) se tiene que:

$$A \times B = \{(J, \color{blue}{\heartsuit}), (J, \color{black}{\spadesuit}), (Q, \color{blue}{\heartsuit}), (Q, \color{black}{\spadesuit}), (K, \color{blue}{\heartsuit}), (K, \color{black}{\spadesuit})\}$$
$$B \times A = \{(\color{blue}{\heartsuit}, J), (\color{blue}{\heartsuit}, Q), (\color{blue}{\heartsuit}, K), (\color{black}{\spadesuit}, J), (\color{black}{\spadesuit}, Q), (\color{black}{\spadesuit}, K)\}$$
