## Diferencia

Diferencia: Definición

Sean \(A\) y \(B\) dos conjuntos, el conjunto diferencia de \(A\) y \(B\) es el formado con los elementos que están en \(A\) y no están en \(B\). El conjunto diferencia es único, se nota \(A \setminus B\) y se define formalmente mediante el siguiente predicado:
$$\phi_{A \setminus B} (x) \Leftrightarrow x \in (A \setminus B) := (x \in A) \land (x \notin B)$$

Utilizando notación por comprensión
$$A \setminus B = \{x : (x \in A) \land (x \notin B)\}$$

Diferencia: Propiedades

Sean \(A\) y \(B\) conjuntos, \(U\) un universo para \(A\) y \(B\).

Propiedad Nombre
\(A \setminus \emptyset = A\) Identidad
\(A \setminus A = \emptyset\) Nilpotencia
\(A \setminus B \neq B \setminus A\) No conmutatividad

Diferencia: Propiedades

Sean \(A\) y \(B\) conjuntos, \(U\) un universo para \(A\) y \(B\).

Propiedad Nombre
\(A \setminus B = A \cap \bar B\)
\(U \setminus A = \bar A\)
\(U \setminus \bar A = A\)
\(A \setminus B = \emptyset \Leftrightarrow A \subseteq B\)
\(A \setminus B = A \Leftrightarrow A \cap B = \emptyset\)
