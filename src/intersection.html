## Intersección

Intersección: Definición

Sean \(A\) y \(B\) dos conjuntos, el conjunto intersección de \(A\) y \(B\) es el formado con los elementos que están en \(A\) y están en \(B\). El conjunto intersección es único, se nota \(A \cap B\) y se define formalmente mediante el siguiente predicado:
$$\phi_{A \cap B} (x) \Leftrightarrow x \in (A \cap B) := (x \in A) \land (x \in B)$$

Utilizando notación por comprensión
$$A \cap B = \{x : (x \in A) \land (x \in B)\}$$

Si \(A \cap B = \emptyset\) se dice que los conjuntos \(A\) y \(B\) son disyuntos o disjuntos.

Intersección: Propiedades

Sean \(A\), \(B\) y \(C\) conjuntos.

Propiedad Nombre
\(A \cap B = B \cap A\) Conmutativa
\((A \cap B) \cap C = A \cap (B \cup C)\) Asociativa
\(A \cap B \subseteq A\)
\(A \cap B \subseteq B\) 		Absorción
\(A \cap \emptyset = \emptyset\) Dominación
\(A \cap A = A\) Idempotencia

Intersección: Propiedades

Sean \(A\), \(B\) y \(C\) conjuntos.

Propiedad Nombre
\(A \subseteq B \Leftrightarrow A \cap B = A\)

Unión e Intersección: Propiedades

Sean \(A\), \(B\) y \(C\) conjuntos.

Propiedad Nombre
\(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\)
\(A \cup (B \cap C) = (A \cup B)\cap (A \cup C)\) 		Distributiva
